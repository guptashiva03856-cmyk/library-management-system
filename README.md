# 📚 Library Management System

## 🚀 Overview

This project is a Library Management System developed during my internship.
It allows managing Authors, Books, and Members using a Python-based backend and simple frontend interface.

---

## ✨ Features

* Manage Authors (Add, Update, Delete)
* Manage Books (Add, Update, Delete)
* Manage Members (Add, Update, Delete)

---

## 🛠️ Tech Stack

* Python
* HTML, CSS, JavaScript
* JSON (for data storage)

---

## 📁 Project Structure

* backend/
* frontend/
* data/

---

## ⚙️ How to Run

```bash
pip install -r requirements.txt
python backend/app.py
```

---

## 🔌 API Endpoints

* GET /authors
* POST /authors
* PUT /authors/{id}
* DELETE /authors/{id}

(Similar for Books and Members)

---

## 🎯 Purpose

This project demonstrates basic backend architecture and CRUD operations.
